U.S. stock returns across every starting month since 1926.

Pick a contribution schedule. See how it would have played out over every rolling N-year window in U.S. equity history — terminal values, distributions, best and worst decades, all in nominal and real terms. Each scenario ships an interactive page and a downloadable PDF report.

Scenarios

Each scenario applies a different contribution schedule to the same monthly returns from the Ken French Data Library and FRED CPI. The same math, different ways of deploying capital — useful for seeing what timing actually buys (or costs) you.

Buy & hold

$1,000 lump sum

Invest $1,000 at the start of each rolling N-year window and hold to the end. The classic buy-and-hold benchmark. Reports annualized CAGR across every starting month since 1926, in horizons of 5, 10, 15, and 20 years.

View analysis → PDF report
Dollar-cost averaging

$100 annual DCA

Invest $100 on each yearly anniversary of the window start — one deposit per year of the horizon, with total contributions scaling from $500 (5-year) to $2,000 (20-year). Reports money-weighted IRR. Comparing this against the lump-sum CAGR shows how much the timing of deployed capital matters.
