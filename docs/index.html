U.S. Stock Returns

Rolling-window analysis of U.S. total-market returns since 1926. Data sources: Ken French Data Library (value-weighted total market) and FRED CPIAUCNS (inflation).

Controls
Annualized return (CAGR) across rolling windows
Each point is the annualized return of a window starting at that date.
Distribution of rolling CAGRs
Terminal value of $1,000
Value at the end of the window, for $1,000 invested at the start.
Summary
Methodology: monthly returns compounded over rolling N-year windows; CAGR = (terminal/1)1/N − 1. Real returns deflate by month-over-month CPI. Source code: repository.
