U.S. Stock Returns

Rolling-window analysis of U.S. total-market returns since 1926. Each scenario applies a different contribution schedule to the same monthly returns and reports terminal values and annualized returns across every rolling window. Data sources: Ken French Data Library (value-weighted total market) and FRED CPIAUCNS (inflation).

Scenarios
  • $1,000 lump sum — Invest $1,000 at the start of each rolling N-year window and hold to the end. The classic buy-and-hold view. Metric: annualized CAGR.
    Download PDF report
  • $100 annual DCA — Invest $100 on each yearly anniversary of the window start for ten years ($1,000 total contributed). Metric: money-weighted IRR. Comparing this scenario's IRR to the lump-sum CAGR shows how much the timing of deployed capital matters.
    Download PDF report
